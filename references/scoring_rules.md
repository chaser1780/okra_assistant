# 默认评分规则

使用这套规则作为第一版研究打分框架，再按真实数据源和基金类型微调。

## 动作区间

- `add`：`score >= 70`，且没有硬性风险事件
- `observe`：`55 <= score < 70`
- `reduce`：`40 <= score < 55`
- `avoid`：`score < 40`，或出现重大负面风险

## 基础维度

- 行情强弱：优先看日涨跌、近一周、近一月
- 新闻影响：按 `positive / neutral / negative` 做加减分
- 风险画像：保守型账户对高波动主题基金额外扣分
- 数据完备度：缺少新闻或缺少基准信息时降低置信度

## 基金类型补充

- 指数基金 / ETF：增加跟踪误差、成交额、费率、折溢价检查
- 主动权益：增加基金经理稳定性、回撤、风格漂移和规模变化
- 债基：增加久期、信用风险、杠杆水平和大额赎回压力
- QDII：增加汇率、时差、海外市场休市与新闻滞后判断

## 输出要求

- 每只基金都给出动作、分数、依据、风险、缺口
- 不把新闻情绪当成唯一结论来源
- 不输出脱离风险画像的绝对化买卖指令
