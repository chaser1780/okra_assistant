- 文档定位：桌面端界面改版后的验收与回归基线

## 1. 本次验收范围

本次验收覆盖：

- Dashboard 首屏决策面板与 Portfolio / Risk Cockpit
- Runtime 任务指挥台、通知中心、日志区
- Realtime 专业监控表、筛选、排序、结构化详情
- Research / Agents 搜索筛选与结构化详情
- Review 图形化摘要与报告分栏
- Trade 交易前检查台与结果区
- Settings 观察池筛选、系统健康结构化视图
- UI 状态持久化、分栏记忆、页内通知流、统一主题 token

## 3. 页面验收清单

### Dashboard

- [x] 首屏能直接回答“今天要不要动”
- [x] 今日要做 / 可以等 / 不建议做三块信息均在首屏
- [x] 风险与可信度信息不再埋在长文本中
- [x] 组合、暴露、执行、健康状态可在右侧 Cockpit 联查
- [x] 最近通知可从首页直接回看

### Runtime

- [x] 手动运行入口集中在任务指挥台
- [x] 当前任务、当前步骤、自动刷新状态位于同一视野
- [x] 任务日志可滚动查看
- [x] 非阻塞通知可回看最近历史

### Realtime

- [x] 支持按代码/名称搜索
- [x] 支持按状态、模式筛选
- [x] 支持按收益、涨跌、可信度排序
- [x] 陈旧数据与正负收益有明确视觉区分
- [x] 右侧详情为结构化卡片，不再只有长文本

### Research

- [x] 支持按动作、执行状态、代码/名称筛选
- [x] 列表直接显示动作、金额、执行状态
- [x] 详情首屏可见可信度、执行状态、建议金额
- [x] 依据、风险、规则校验已分节展示

### Agents

- [x] 支持按状态、名称筛选
- [x] 列表直接显示状态和置信度
- [x] 详情可快速识别摘要、关键点、缺失信息、观察项

### Review

- [x] 左侧有统计与历史表现图形化摘要
- [x] 支持按来源和周期筛选
- [x] 最新 lessons / bias adjustments / agent feedback 独立呈现
- [x] 右侧保留完整报告或 fallback 详情

### Trade

- [x] 左侧为分组表单，不再是单一堆叠录入区
- [x] 右侧交易前检查台会即时刷新
- [x] 现金、上限、可卖金额、持有约束已显式检查
- [x] 交易结果与流水独立显示

### Settings

- [x] 快捷入口位于第一屏
- [x] 策略控制和单只上限分区明确
- [x] 观察池支持筛选与结构化编辑
- [x] 系统健康、manifest、自检和路径为结构化视图

## 4. 交互与一致性清单

- [x] 标签页命名已统一为中文
- [x] 数值字体和正文层级已区分
- [x] 状态语义通过 badge 和色彩统一表达
- [x] 保存、完成、失败提示优先使用页内通知
- [x] 删除观察池保留确认弹窗
- [x] 页面在无数据时会显示空状态说明
- [x] 分栏位置、筛选条件、上次标签页、部分选中项会持久化
